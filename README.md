#### 版本：v1.0.0
#### 网站：http://www.python-pro.com
#### 使用说明：
+ 下载zip文件
+ 在head部分引入css文件
  - &lt;link href="gallery.css" rel="stylesheet"&gt;
+ 在head部分引入js文件
  - <script src="jquery.js"></script>
  - <script src="gallery.js"></script>
+ 为需要lightbox效果的图片添加 class="ViewImg" 属性
